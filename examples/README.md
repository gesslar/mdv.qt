# mdv examples

These documents exist to exercise — and eyeball — everything mdv styles. Open
them in mdv and switch content themes (File → Preferences) to see how each
theme renders the full range of elements and code.

- **[elements.md](elements.md)** — every Markdown element mdv themes:
  headings, text emphasis, lists, tables, blockquotes, inline code, links,
  rules.
- **[syntax-highlighting.md](syntax-highlighting.md)** — fenced code blocks
  across many languages, plus the "no highlighting" cases (`text`, `none`,
  and an unlabeled fence).

If you add a new style slot to the content theme or a new highlighting rule,
add a case here so it stays testable.
